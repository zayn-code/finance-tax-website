# 智慧财税官网

一个专业的财税服务公司官网，基于 Next.js 14 构建。

## 🌟 特性

- ✅ **现代化技术栈**：Next.js 14 + TypeScript + Tailwind CSS
- ✅ **响应式设计**：完美适配桌面端和移动端
- ✅ **SEO 优化**：完整的 meta 标签、sitemap、robots.txt
- ✅ **高性能**：服务器端渲染 + 静态生成
- ✅ **交互功能**：联系表单、在线咨询

## 📱 页面结构

- **首页**：企业介绍、核心优势展示
- **服务项目**：6大核心服务详细介绍
- **关于我们**：公司简介、团队介绍、发展历程
- **联系我们**：多种联系方式、在线表单、办公地址

## 🚀 快速开始

### 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问 http://localhost:3000
```

### 构建生产版本

```bash
# 构建
npm run build

# 启动生产服务器
npm start
```

## 📦 部署指南

### Vercel 部署（推荐）

1. 注册 [Vercel](https://vercel.com) 账号
2. 连接 GitHub 仓库
3. 导入项目，自动部署

### 传统服务器部署

```bash
# 构建项目
npm run build

# 使用 PM2 管理进程
npm install -g pm2
pm2 start npm --name "finance-website" -- start
```
